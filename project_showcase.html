Vezi și celelalte proiecte ale mele

WEATHER APP

O aplicație meteo responsivă. Această aplicație permite utilizatorilor să caute condiții meteo în timp real după numele orașului, folosind serviciul WeatherAPI.

CHESS GAME

Un joc de șah complet bazat pe interfață grafică, implementat în Python folosind tkinter și puternicul motor python-chess pentru validarea mutărilor și gestionarea stării jocului.

TIC-TAC-TOE GAME

Un joc de X și 0 complet interactiv pentru doi jucători, construit folosind Pygame. Acest proiect demonstrează gestionarea evenimentelor, randarea grafică a grilelor și implementarea logicii pe ture.

WEB TIC-TAC-TOE

Un joc curat, responsiv (X și O), construit folosind HTML, CSS și JavaScript Vanilla. Fără biblioteci — doar front-end pur.

TETRIS GAME

Acesta este un joc clasic Tetris implementat în Python folosind biblioteca Pygame. Include toate mecanismele de bază, cum ar fi rotirea formelor, detectarea coliziunilor, curățarea rândurilor și creșterea nivelului de dificultate.

SHOOTING ASTEROIDS

Acest proiect este un joc arcade bazat pe Python, construit folosind Pygame. Jucătorul controlează o navă și trebuie să supraviețuiască valurilor de asteroizi. Jocul include animații, detectarea coliziunilor, urmărirea scorurilor și a vieților.

SNAKE GAME

Acest proiect este o abordare modernă a jocului clasic Snake, implementat folosind Python și Pygame. Jocul prezintă dificultate dinamică, cu niveluri multiple și configurații de obstacole, care evoluează odată cu acumularea de puncte.

PING-PONG GAME

Un joc clasic de ping-pong pentru doi jucători, construit folosind Python și Pygame. Jocul oferă controale fluide ale paletei, fizica dinamică a mingii și urmărirea scorului în timp real.

MATCHING CARD NUMBERS GAME

Acesta este un joc clasic memorare, dezvoltat cu Python și Pygame. Jocul se bazează pe găsirea de perechi de cărți potrivite. Oferă o interfață grafică curată și mecanici de interacțiune receptive.

PROTOTYPE - MY FIRST PORTFOLIO WEBSITE

Acest proiect arată punctul meu de plecare în dezvoltarea web. Acest website stă la baza a ceea ce se găsește acum pe internet. Prototipul de la care mi-am dezvoltat actualul portofoliu online.

PASSWORD GENERATOR

Un generator de parole modern și curat. Acesta permite utilizatorilor să genereze parole aleatorii și sigure, cu opțiuni configurabile pentru lungime și tipuri de caractere.

WEB QR CODE GENERATOR

Un generator de coduri QR curat și simplu, construit folosind biblioteca qrcode prin CDN. Nu sunt necesare instrumente de compilare.

TO-DO LIST

O aplicație web simplă și responsivă pentru liste de activități. Activitățile pot fi adăugate, editate, marcate ca finalizate, filtrate și șterse — totul în browser.

CALCULATOR APP

Un calculator complet funcțional, suport pentru tastatură și sunete la clicuri. Conceput pentru a fi curat, responsiv și ușor de utilizat.

DIGITAL CLOCK

Un ceas digital simplu. Ceasul afișează ora, minutul și secunda curente, cu o animație de scalare lină la fiecare actualizare.

APROZAR DIGITAL

„Aprozar Digital” un prototip de platfomă online ce se ocupă cu comercializarea fructelor și legumelor. Acest proiect este un exercițiu care mi-a consolidat cunoștințele în Python și Django.

Warehouse Manager

Proiectul reprezintă un sistem de gestionare a depozitului, conceput pentru a simula operațiuniledin lumea reală. Include suport pentru mai multe categorii de produse, rezervări, achiziție, etc..
